# 📦 Almoxarifado Melhoramentos App

Sistema desenvolvido para auxiliar no controle e gerenciamento de materiais de almoxarifado, proporcionando maior organização, rastreabilidade e eficiência na movimentação de estoque.

O projeto foi criado com foco na digitalização de processos internos, reduzindo controles manuais e facilitando o acompanhamento de entradas, saídas e disponibilidade de materiais.

---

## 🎯 Objetivo

Centralizar o gerenciamento de materiais e itens de estoque em uma única aplicação, permitindo maior controle operacional e melhor tomada de decisão.

---

## 📦 Instalação

Clone o repositório:

```bash
git clone https://github.com/AdautoFurich/almoxarifado-melhoramentos-app.git
```

Entre na pasta:

```bash
cd almoxarifado-melhoramentos-app
```

Instale as dependências:

```bash
flutter pub get
```

Execute o aplicativo:

```bash
flutter run
```

---

## 🔨 Gerando APK

Para gerar a versão Android:

```bash
flutter build apk --release
```

APK gerado em:

```text
build/app/outputs/flutter-apk/app-release.apk
```
